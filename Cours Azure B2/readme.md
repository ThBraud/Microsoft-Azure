# Informations sur le projet  
- Projet réalisé entre le 23 et 27 février 2026 en cours  
- Fait sur Microsoft Azure avec la licence étudiante
- Ce TP avait pour but de nous faire découvrir Azure  

## But du TP 
- Création d'une VM sur Azure
- Création d'un serveur SQL
- Utilisation/découverte de Docker
- Utilisation d'Azure DevOps et de DockerHub
- Création d'une pipeline et d'un runner
- Avoir une page internet fonctionnelle
